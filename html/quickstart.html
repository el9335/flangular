This section describes how to install and run the server, and provides instructions for interacting with the web application.

Flangular Server Install and Run

This tutorial assumes that your system has pip and easy_install installed, as they are required to install Flangular dependencies. This version requires that the server be run on the same machine as the web application.

1. Clone the Flangular repository at https://github.com/el9335/flangular.git.

2. Enter the Flangular directory using cd flangular

3. Build the app using sudo python setup.py install, or run as administrator on Windows (Windows is NOT natively supported by this application beyond calling the hosting system's separator for file paths.)

4. Start the server using sudo python flangular.py. You should see the following output (or similar):

Building the Grammar Model
Time: (9.69)s
* Restarting with stat
* Debugger is active!
* Debugger pin code: 264-454-349

The Flangular server must be running in order for the web application to work as intended.
