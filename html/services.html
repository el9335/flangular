The following low-level Natural Language Processing services are currently available via the Flangular server.

  • Tokenization: tokens
    The process of breaking a text into its constituent words.
    Resource: Python Natural Language Tool Kit http://www.nltk.org/

  • Sentence Boundary Detection: sentence
    The process of breaking a text into its constituent sentences.
    Resource: Python Natural Language Tool Kit http://www.nltk.org/

  • Part-of-Speech Tagging: POSTags
    The process of tagging each token in a text with its part of speech. See the Penn Tree Bank tab for a list of labels and definitions.
    Resource: Python Natural Language Tool Kit http://www.nltk.org/

  • Dependency Parsing: ParseTree
    The process of determining the grammatical structure of the sentence. See the Penn Tree Bank tab for a list of labels and definitions.
    Resource: pyStatParser https://github.com/emilmont/pyStatParser
